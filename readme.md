# 🐾 petshop vida mansa - fase 2

este é o projeto da **fase 2** da disciplina de desenvolvimento web. o objetivo foi criar um site de petshop moderno e interativo.

### 🚀 o que tem de novo:
- **carrossel:** banners rotativos com as promoções da loja.
- **saudação dinâmica:** mensagem automática que muda conforme a hora do dia.
- **agendamento:** formulário completo com seleção de data e hora.
- **acessibilidade:** descrições detalhadas em todas as imagens.

### 🛠️ tecnologias:
- html5 / css3
- bootstrap 5
- javascript (front-end)
